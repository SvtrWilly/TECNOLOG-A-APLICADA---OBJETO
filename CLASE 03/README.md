# Descripción del proyecto

- Este proyecto explora la traducción de datos fisiológicos en una experiencia visual sensible.
A partir del registro de ritmo cardiaco obtenido desde un smartwatch, se construye una visualización
en Processing que transforma estos valores en una forma orgánica en constante cambio.

- En lugar de representar los datos como un gráfico tradicional, se propone una interpretación
donde el pulso se manifiesta como un cuerpo dinámico: una figura circular que se expande, contrae
y deforma suavemente según las variaciones del ritmo cardiaco. La forma mantiene una coherencia
estructural, evitando rupturas abruptas, para enfatizar una sensación de continuidad y respiración.

- El color actúa como una extensión de este comportamiento. A medida que la forma crece, transita
hacia tonos cálidos (rojos), mientras que en estados más contraídos adopta tonalidades frías (azules),
estableciendo una relación directa entre tamaño y estado visual.

- De esta manera, el proyecto desplaza el dato desde su condición numérica hacia una dimensión perceptiva,
permitiendo observar el ritmo cardiaco no solo como medición, sino como una manifestación visual de procesos internos del cuerpo.
