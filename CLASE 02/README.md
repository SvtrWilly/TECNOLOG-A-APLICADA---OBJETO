# CLASE 02
## Escribir 10 cualidades de 2 objetos

## Acuario y Guitarra

## Acuario

Contiene agua
Alberga peces u organismos vivos
Material principal: vidrio
Transparente
Funciona como contenedor
Requiere electricidad (filtro, luz, oxigenación)
Genera reflejos y distorsiones visuales
Tiene límites definidos (interior/exterior)
Mantiene un ecosistema controlado
Tiene un tamaño establecido

## Guitarra

Fabricada principalmente de madera
Tiene acabado de barniz
Contiene elementos metálicos (cuerdas, trastes)
Posee trastes que segmentan el sonido
Tiene cuerdas tensadas
Produce sonido mediante vibración
Puede ser acústica o eléctrica
Color predominante: negra y roja
Requiere interacción humana para sonar
Tiene una forma ergonómica adaptada al cuerpo
